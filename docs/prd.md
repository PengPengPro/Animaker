# 漫画生成器需求文档

## 1. 应用概述

### 1.1 应用名称
漫画生成器

### 1.2 应用描述
漫画生成器是一个专业的在线创作工具,帮助用户将创意故事转化为专业的漫画和动态视频作品。用户可以输入故事框架,通过AI自动生成分镜头脚本,编辑完善后生成精美的漫画图片,并进一步转换为动态视频,完成从故事到作品的完整创作流程。

## 2. 核心功能

### 2.1 故事创作流程
- 在首页输入故事框架
- AI自动生成专业的分镜头脚本
- 支持实时编辑和修改脚本内容
- 每个分镜都有精炼的描述
- 支持多个分镜的管理

### 2.2 漫画生成
- 根据分镜脚本生成漫画图片
- 支持多种艺术风格选择:
  - 美漫风格
  - 日漫风格
  - 韩漫风格
  - 中国古风
  - 手绘线条
- 每个分镜可单独生成漫画
- 支持预览生成的漫画图片
- 支持下载漫画图片
- 漫画以上下连续的方式展示
- 点击漫画可以查看大图

### 2.3 动态视频生成
- 将静态漫画转换为动态视频
- 自动生成视频缩略图
- 支持视频预览
- 支持视频下载
- 预览区分为2列:左面为生成的漫画,右边为生成的视频
- 视频支持预览和导出
- 默认生成5秒时长、720p分辨率的动态视频
- 动态化流程:
  - 确认漫画已生成
  - 点击分镜脚本上的动态化按钮
  - 提交动态化申请
  - 超级管理员审核申请
  - 审核通过后开始生成视频
  - 显示生成进度
  - 生成完成后自动显示视频

### 2.4 作品管理
- 左侧作品列表,可折叠/展开
- 支持创建、查看、切换和删除作品
- 所有作品自动保存,永不丢失
- 支持创建多个作品项目
- 可以随时切换和管理不同的作品
- 支持删除不需要的作品
- 作品名称可编辑并自动保存
- 作品列表收起时,预览区自动拉伸

### 2.5 用户系统
- 账号密码注册登录功能
- 未登录时点击首页的生成脚本出现登录弹窗
- 未登录时点击作品出现登录弹窗
- 已登录时侧边栏显示用户名
- 已登录时点击侧边栏底部出现退出登录按钮
- 第一个注册的用户自动成为管理员
- 超级管理员权限:
  - 审核动态化视频生成申请

## 3. 界面结构

### 3.1 侧边栏
- 首页
- 作品
- 底部折叠/展开按钮
- 底部账号/用户名显示

### 3.2 作品页布局
- 左侧:作品列表(可折叠)
- 中央:分镜编辑区和风格设置面板
- 右侧:预览区(双列布局)

### 3.3 预览区
- 左列:漫画展示
- 右列:视频展示
- 漫画和视频使用相同的交互方式
- 鼠标悬停显示操作按钮
- 点击图片选中对应分镜脚本

## 4. 技术说明

### 4.1 AI集成
- 文本生成大模型:生成分镜头脚本
- 图片生成API:生成漫画图片
- 图生视频API:生成动态视频

### 4.2 数据存储
- 使用Supabase数据库存储作品和分镜数据
- 支持用户认证和个人作品管理
- 视频生成申请和审核状态持久化存储

### 4.3 状态管理
- 实时显示生成进度和状态
- 完善的错误处理机制
- 友好的错误提示

## 5. 交互说明

### 5.1 作品列表控制
- 侧边栏底部按钮:点击可折叠/展开作品列表
- 顶部展开按钮:作品列表折叠时显示
- 标题栏收起按钮:作品列表展开时显示

### 5.2 预览交互
- 漫画和视频 hover 时显示操作按钮
- 点击漫画查看大图
- 点击视频预览播放
- 支持下载漫画和视频

### 5.3 动态化流程
- 确认漫画已生成
- 点击分镜脚本上的动态化按钮
- 提交动态化申请
- 显示审核提示信息
- 超级管理员审核通过后开始生成视频
- 显示生成进度
- 生成完成后自动显示视频
- 支持预览和下载

## 6. 其他说明

### 6.1 图片处理
- 自动将图片URL转换为Base64编码
- 验证图片格式(.jpg/.jpeg/.png)
- 检查文件大小(不超过10MB)
- 确保符合API规范要求

### 6.2 视频生成参数
- 图生视频的prompt为自动带入的{脚本},其中{脚本}为image对应的分镜脚本
- 默认视频时长:5秒
- 默认视频分辨率:720p

### 6.3 作品页布局
- 作品标题与分镜脚本标题居左对齐
- 侧边栏显示Comi品牌logo(64×23像素)
- 布局简洁,视觉层次清晰
